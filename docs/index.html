3. Welche Daten sammelt die App?

Poker Guessr sammelt keine persönlichen Daten.

  • Es werden keine Benutzerkonten erstellt.
  • Spielernamen werden nur lokal auf dem Gerät während einer Spiel-Session gespeichert und nicht an Server übermittelt.
  • Es werden keine Standortdaten erhoben.
  • Es wird kein Tracking oder Analytics verwendet.
  • Es werden keine Daten an Dritte weitergegeben.

4. Netzwerkzugriff

Die App nutzt Google Firebase Firestore ausschließlich zum Laden von Quiz-Fragen. Dabei werden keine personenbezogenen Daten übermittelt.

5. Drittanbieter-Dienste

Die App verwendet:

  • Firebase Firestore (Google) – zum Laden der Quiz-Inhalte. Firebase Datenschutz

Es werden keine Werbe-SDKs, Analytics-Tools oder Crash-Reporter eingesetzt.

6. Datenspeicherung

Alle App-Einstellungen werden lokal auf dem Gerät über UserDefaults gespeichert. Diese Daten verlassen das Gerät nicht.

7. Kinder

Die App sammelt keine Daten und ist für alle Altersgruppen geeignet (4+).

Diese Datenschutzerklärung kann bei Bedarf aktualisiert werden. Die aktuelle Version ist immer unter dieser URL verfügbar.
